Discord Code Watcher

Bridge: —

Channel IDs: one or more, comma-separated. Right-click a channel → "Copy Channel ID". Open each channel in its own Discord tab (same Chrome profile) so all are watched at once.
Watch user ID: the stable Discord user ID (recommended). Right-click the user → "Copy User ID" (enable Developer Mode in Discord settings first).
Fallback names: used only when no user ID matches — handy if you don't have the ID. Matched case-insensitively, but names change when the user renames.
